### About Me

Full-stack engineer who ships products and obsesses over performance. I build multi-market e-commerce platforms serving users across **11 languages**, optimize systems at the byte level, and write about what I learn.

- Architecting high-traffic e-commerce with **Vue/Nuxt**, **Next.js**, **PostgreSQL**, and **Redis**
- Cut API response sizes by **60%**, achieved **6x cache capacity**, delivered **80% faster search**
- Building **Juicify** — AI-powered fitness platform
- Cooperating with **Deante** — multi-market digital presence for a leading manufacturer
- Digital nomad. Building from anywhere.
